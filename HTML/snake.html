Snake

Control:

Go Up: Arrow Up

Go Down: Arrow Down

Go Left: Arrow Left

Go Right: Arrow Right

To Start: Press an Arrow Key

Instructions:

You have to try to eat as many mice (gray) as possible.

When it has eaten a mouse, the snake grows.

After every fifth mouse the snake gets faster.

The game ends when the snake hits the wall or itself.
